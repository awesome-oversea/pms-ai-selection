# Repository Description

## Short Description

AI-powered product selection PMS for cross-border e-commerce with multi-role workbench, approval workflow and local ERP profit feedback loop.

## Chinese Description

跨境电商 AI 选品 PMS：覆盖选品推荐、人工审批、采购执行、利润反馈、报告中心和多角色工作台的本地可运行 MVP。

## About Section

`A portfolio MVP showing how AI product selection can be embedded into a real cross-border e-commerce workflow: trend/event signals -> GO/NO-GO decision -> human approval -> SCM/WMS/OMS execution -> profit feedback -> rescore and report.`

## Suggested README Opening

`This project demonstrates a business-first AI product selection system for cross-border e-commerce. Instead of stopping at model output, it connects recommendations to approval, procurement execution, profit feedback and report delivery.`
