本文档按“可直接执行”的口径整理当前项目的核心测试用例与验收脚本，优先覆盖：

- 异常路径
- 治理与交付巡检
- 页面展示与交付结果

## 9. 判定标准

若以下全部满足，则当前可判定为“本地交付验收通过”：

- 主链正常路径通过
- 主链异常路径通过
- 数据采集场景验收通过
- Flink checkpoint 本地链路通过
- 配置治理通过
- RAG治理通过
- delivery readiness = ready
- delivery conclusion = delivery_ready
- delivery scope boundary 已生成
