## 4. 环境检查

### 4.1 K8s集群检查

```bash
#!/bin/bash
# K8s集群健康检查脚本

echo "=== K8s集群健康检查 ==="

# 检查节点状态
echo "1. 节点状态:"
kubectl get nodes -o wide

# 检查核心组件
echo "2. 核心组件:"
kubectl get pods -n kube-system

# 检查命名空间
echo "3. 业务命名空间:"
kubectl get namespaces | grep fms

# 检查存储
echo "4. 存储状态:"
kubectl get pvc -A

# 检查GPU
echo "5. GPU节点:"
kubectl describe nodes -l node-role.kubernetes.io/worker-gpu | grep -A 5 "Allocatable"
```
