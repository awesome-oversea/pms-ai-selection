## 3. 关键配置

```conf
# redis.conf
port 6379
cluster-enabled yes
cluster-config-file nodes.conf
cluster-node-timeout 5000
appendonly yes
appendfsync everysec
maxmemory 4gb
maxmemory-policy allkeys-lru
save 900 1
save 300 10
save 60 10000
requirepass <encrypted>
masterauth <encrypted>
```

## 4. 缓存用途规划

| Key前缀 | 用途 | TTL | 数据量(估) |
|---------|------|-----|----------|
| `session:` | 用户会话 | 30min | ~1000条 |
| `cache:query:` | 查询缓存 | 5min | ~50000条 |
| `cache:llm:` | LLM响应缓存 | 1h | ~10000条 |
| `rate:` | 限流计数器 | 1min | ~5000条 |
| `lock:` | 分布式锁 | 30s | ~100条 |
| `agent:state:` | Agent状态 | 10min | ~500条 |
