## 2. Kubernetes网络规划

### 2.1 IP地址段规划

| 网络类型 | CIDR | 用途 | 容量 |
|---------|------|------|------|
| VPC | 10.0.0.0/16 | 虚拟私有云 | 65536 IP |
| 应用子网-AZ-A | 10.0.1.0/24 | Master + Worker节点 | 254 IP |
| 应用子网-AZ-B | 10.0.2.0/24 | Master + Worker节点 | 254 IP |
| 应用子网-AZ-C | 10.0.3.0/24 | Master + Worker节点 | 254 IP |
| GPU子网 | 10.0.100.0/24 | GPU计算节点 | 254 IP |
| 数据库子网 | 10.0.200.0/24 | PG/Redis/Qdrant | 254 IP |
| Pod CIDR | 10.244.0.0/16 | Pod IP地址池 | 65536 IP |
| Service CIDR | 10.96.0.0/12 | Service VIP池 | 1048576 IP |

### 2.2 CNI网络方案

| 配置项 | 选型/值 | 说明 |
|--------|---------|------|
| CNI插件 | Calico v3.26+ | 支持NetworkPolicy + BGP |
| IP模式 | BGP | 跨节点Pod通信，无需封装 |
| IPAM | Calico IPAM | 每节点/26子网(64 Pod IP) |
| MTU | 1500 | 标准以太网MTU |
| NetworkPolicy | 默认拒绝入站 | 最小权限原则 |

## 4. 安全网络策略

### 4.1 默认拒绝策略

```yaml
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: default-deny-all
  namespace: pms-api
spec:
  podSelector: {}
  policyTypes:
    - Ingress
    - Egress
```

### 4.2 命名空间间通信规则

| 源命名空间 | 目标命名空间 | 允许端口 | 说明 |
|-----------|------------|---------|------|
| pms-api | fms-data | 5432, 6379 | API→数据库/缓存 |
| pms-api | pms-agent | 8100 | API→Agent调用 |
| pms-agent | pms-llm | 8300 | Agent→LLM推理 |
| pms-agent | pms-rag | 8200 | Agent→RAG检索 |
| pms-rag | fms-data | 5432, 6333 | RAG→PG/Qdrant |
| pms-monitoring | ALL | 9090 | Prometheus指标采集 |
