## 验收总则

1. 每项任务验收须同时满足**功能验收**和**质量验收**两部分
2. 功能验收：逐条检查功能点是否实现，全部 🟢 项通过方可标记为 ✅已完成
3. 质量验收：代码质量、测试覆盖、文档完整性，🟡 项不通过可标记为 🔧进行中
4. 已完成任务（原清单标记 ✅已完成）需通过回归验证确认无退化
5. 所有 API 端点须通过自动化测试或 curl/Postman 手动验证
6. 安全相关任务须通过安全扫描（ruff + mypy + 手动审查）
