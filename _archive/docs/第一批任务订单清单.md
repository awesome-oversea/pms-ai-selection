## 3. 任务订单详情

## T01 智能选品统一决策输出层

- 任务说明：
  将 `DataCollection / MarketInsight / ProductPlanner / Commercial` 的阶段输出收敛为统一结构化决策对象，供选品结果 API、BFF、BI、后续飞轮能力稳定消费。
- 实现范围：
  - `src/agents/selection_master.py`
  - `src/services/selection_service.py`
  - `src/models/schemas.py`
  - `src/api/v1/endpoints/selection.py`
  - `src/api/v1/endpoints/bff.py`
  - 相关测试
- 依赖：无
- 验收标准：
  1. 选品结果中存在统一决策对象。
  2. 决策对象至少包含：推荐结论、推荐理由、风险项、利润项、定价项、供应链项、证据来源、待确认事项。
  3. 任务详情/BFF 可直接消费统一结构。
  4. 至少补充 1 组服务层测试和 1 组接口/BFF 测试。
- 执行状态：已完成
- 验收结果：通过。统一决策对象已落地到 SelectionMaster / SelectionTaskService / BFF；关键验收测试 3 项通过。

---

## T02 利润 / ROI / 定价优化引擎正式化

- 任务说明：
  将现有利润优化从样板结果升级为正式业务输出，支持显式成本项输入与保守/基准/激进三档场景分析。
- 实现范围：
  - `src/services/profit_optimization_service.py`
  - `src/api/v1/endpoints/commercial.py`
  - 与 T01 统一决策层的衔接
  - 相关测试
- 依赖：T01
- 验收标准：
  1. 支持至少 3 类成本项输入。
  2. 返回至少 3 档场景分析结果。
  3. 统一决策层可消费利润 / ROI / 定价输出。
  4. 至少补充 1 组服务测试和 1 组接口测试。
- 执行状态：已完成
- 验收结果：通过。已支持显式多成本项输入、conservative/base/aggressive 三档场景分析，并提供 `decision_ready_summary`；关键验收测试 2 项通过。
