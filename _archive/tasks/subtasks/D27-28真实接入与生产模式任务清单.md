# D27-28 真实接入与生产模式任务清单

> 来源文档: [问题与建议.md](../../问题与建议.md)
> 总清单: [子任务清单.md](../../子任务清单.md)
> 时间范围: D27-D28
> 目标: 让系统从“可模拟运行”走向“可真实接入”，并完成真实模式的开关与验证

## D27 真实数据源试点接入

- 选择至少一个真实 provider 作为试点接入。
- 补齐鉴权、限流、错误处理和降级逻辑。
- 将试点 provider 接入任务主链路或 RAG 相关流程。
- 沉淀真实接入的开发模板和接口规范。

## D28 真实 LLM/RAG 生产模式切换

- 为 LLM、embedding、RAG 查询链路增加真实模式开关。
- 校验配置切换后系统行为、日志和监控是否符合预期。
- 保留 mock 回退，但明确触发条件和告警方式。
- 输出生产模式启用检查清单。

## 产出物

- 至少一个真实 provider 接入结果
- 真实模式开关与配置说明
- 生产模式启用检查清单

## 本段验收

- [ ] 至少一条真实数据接入链路跑通。
- [ ] mock 与 real 可以明确切换并被日志识别。
- [ ] 真实模式启用前的配置与依赖检查项明确。
