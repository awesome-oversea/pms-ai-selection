## D62: T129 销量特征

```java
// Flink SQL - 7天销量窗口
SELECT 
    product_id,
    SUM(sales) AS sales_7d,
    COUNT(*) AS order_count_7d,
    AVG(price) AS avg_price_7d
FROM kafka_source
WHERE event_time BETWEEN NOW() - INTERVAL '7' DAY AND NOW()
GROUP BY product_id, TUMBLE(event_time, INTERVAL '1' HOUR);
```
