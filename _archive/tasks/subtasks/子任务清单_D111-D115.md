  - lint

  - build
  - deploy

  stage: lint
  script:
    - ruff check .
    - mypy --strict src/

  stage: test
  script:
    - pytest tests/ --cov=src/ --cov-report=xml

## D114: T180 自动化部署

**部署策略**:
```yaml
# 灰度发布配置
deployment:
  strategy: canary
  steps:
    - canary_5%:   # 5%流量验证
    - canary_20%:  # 扩展灰度
    - canary_50%:  # 半量发布
    - full:        # 全量上线
  rollback: automatic  # 错误率>1%自动回滚
```
