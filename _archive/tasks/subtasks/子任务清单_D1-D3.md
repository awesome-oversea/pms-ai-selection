## D3 详细任务

### 上午 (09:00-12:00)

#### T003-A: K8s Master节点安装
- [ ] 操作系统初始化（CentOS/Ubuntu）
- [ ] 安装容器运行时（containerd）
- [ ] 安装kubeadm/kubelet/kubectl
- [ ] 初始化第一个Master节点
- [ ] 配置kubectl访问

**命令参考**:
```bash
# 初始化Master
kubeadm init --pod-network-cidr=10.244.0.0/16 \
  --control-plane-endpoint="LOAD_BALANCER_DNS:6443" \
  --upload-certs

# 配置kubectl
mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config
```

### 下午 (13:30-18:00)

#### T003-B: etcd高可用配置
- [ ] 部署外部etcd集群（3节点）
- [ ] 配置etcd证书
- [ ] 加入第二个Master节点
- [ ] 验证集群状态
- [ ] 配置负载均衡器（HAProxy/Keepalived）

**验收标准**:
```bash
# 验证节点状态
kubectl get nodes
# 输出: STATUS=Ready

# 验证组件状态
kubectl get cs
# 输出: 所有组件Healthy
```
