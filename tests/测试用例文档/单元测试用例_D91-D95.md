### 2.7 TestIntegration - 集成测试

#### TC_026: 完整里程碑流程

```python
def test_full_milestone_workflow(self):
    demo = DemoManager()
    review = ReviewManager()

    scenarios = _run_async(demo.init_default_scenarios())
    results = _run_async(demo.execute_all())

    items = _run_async(review.init_phase3_items())
    for item in items:
        passed = random.random() > 0.1
        _run_async(review.evaluate_item(
            item.item_id,
            actual="达标" if passed else "未达标",
            passed=passed,
        ))

    review_result = _run_async(review.create_review(milestone="M3"))
    assert review_result.decision in [ReviewDecision.PASS, ReviewDecision.CONDITIONAL_PASS]

    plan = _run_async(review.create_phase4_plan())
    assert len(plan.objectives) > 0
```

**预期结果**: 完整流程正常工作
