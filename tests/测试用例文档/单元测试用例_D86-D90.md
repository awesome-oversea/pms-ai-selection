### 2.6 TestIntegration - 集成测试

#### TC_027: UAT与Bug追踪完整流程

```python
def test_uat_bug_workflow(self):
    uat = UATManager()
    tracker = BugTracker()

    case = _run_async(uat.create_test_case(...))
    _run_async(uat.execute_test(case.case_id, passed=False, actual_result="功能异常"))

    bug = _run_async(tracker.report_bug(
        title="选品任务创建失败",
        description="无法创建选品任务",
        severity=BugSeverity.HIGH,
        module="选品工作流",
        case_id=case.case_id,
    ))

    _run_async(tracker.update_bug_status(bug.bug_id, BugStatus.FIXED))
    _run_async(tracker.update_bug_status(bug.bug_id, BugStatus.VERIFIED))

    _run_async(uat.execute_test(case.case_id, passed=True, actual_result="已修复"))

    report = _run_async(uat.generate_report())
    assert report.passed_cases >= 1
```

**预期结果**: 完整流程正常工作，报告生成正确
